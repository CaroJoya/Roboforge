# RoboForge 🤖

A social media platform for robotics enthusiasts to share designs, collaborate, and connect.

## Features
- User authentication & profiles
- Post/design sharing with image uploads
- Follow system
- Like & comment on posts
- Messaging system
- Tag-based search
- Notification system
- Saved posts

## Tech Stack
- **Backend**: PHP
- **Database**: MySQL
- **Frontend**: HTML, CSS, JavaScript
- **Icons**: Font Awesome

## Installation

### Prerequisites
- PHP 7.4+
- MySQL 5.7+
- Apache/Nginx web server
